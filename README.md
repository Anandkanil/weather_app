# weather_app
Weather App
Welcome to the Weather App! This simple web application provides real-time weather information for both the user's location and allows them to search for weather details in other cities. The app is built using HTML, CSS, and JavaScript, and it relies on the OpenWeather API to fetch weather data.

Features
Current Location Weather:

Automatically displays the weather information for the user's current location upon loading the app.
City Search:

Allows users to search for weather details in any city around the world.
OpenWeather API Integration:

Utilizes the OpenWeather API to fetch accurate and up-to-date weather data.
